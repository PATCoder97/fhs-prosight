# Task: Update AuthProvider Component

## Description

Cập nhật component `AuthProvider.vue` để chỉ hiển thị 2 OAuth buttons (Google và GitHub) thay vì 4 providers hiện tại. Thay đổi layout từ small icon buttons sang full-width buttons với icon và text, và thêm click handlers để redirect đến backend OAuth endpoints.

- [x] Facebook và Twitter được xóa khỏi authProviders array
- [x] Chỉ còn Google và GitHub providers
- [x] Buttons thay đổi từ icon-only → full-width với icon + text
- [x] Button styling:
  - Google: Icon `tabler-brand-google-filled`, color #db4437, text "Đăng nhập với Google"
  - GitHub: Icon `tabler-brand-github-filled`, color #272727 (light) / #fff (dark), text "Đăng nhập với GitHub"
- [x] Gap 16px giữa 2 buttons
- [x] Click handlers redirect đến `${API_BASE_URL}/api/auth/login/{provider}`
- [x] Dark mode và light mode hiển thị đúng colors
- [x] Responsive trên mobile/tablet/desktop

**New implementation:**
```vue
<script setup>
import { useTheme } from 'vuetify'

const { global } = useTheme()

const authProviders = [
  {
    name: 'Google',
    icon: 'tabler-brand-google-filled',
    color: '#db4437',
    colorInDark: '#db4437',
  },
  {
    name: 'GitHub',
    icon: 'tabler-brand-github-filled',
    color: '#272727',
    colorInDark: '#fff',
  },
]

const loginWithOAuth = (provider) => {
  const baseUrl = import.meta.env.VITE_API_BASE_URL || 'http://127.0.0.1:8001'
  window.location.href = `${baseUrl}/api/auth/login/${provider.toLowerCase()}`
}
</script>

<template>
  <div class="d-flex flex-column gap-4">
    <VBtn
      v-for="provider in authProviders"
      :key="provider.name"
      block
      size="large"
      variant="outlined"
      :color="global.name.value === 'dark' ? provider.colorInDark : provider.color"
      @click="loginWithOAuth(provider.name)"
    >
      <VIcon :icon="provider.icon" start />
      Đăng nhập với {{ provider.name }}
    </VBtn>
  </div>
</template>
```

**Key changes:**
1. Xóa Facebook và Twitter từ array
2. Thêm `name` property cho mỗi provider
3. Thay đổi layout: `flex-column` với `gap-4` (16px)
4. Button props: `block` (full width), `size="large"`, `variant="outlined"`
5. Add `loginWithOAuth` function để handle click
6. Icon position: `start` (trước text)
